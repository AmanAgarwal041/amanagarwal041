## About me

I'm a **Principal Software Engineer and Co-founder** building at the intersection of AI, security, and open-source infrastructure.

At **Transilience AI** I'm building an AI-powered security platform from the ground up — autonomous agents for pentesting, policy creation, compliance, and vulnerability assessment, paired with a dynamic frontend that generates interactive TSX components tailored to each project's AI-generated data. In parallel, I co-founded **OpenLIT**, one of the leading open-source platforms for LLM observability.

Before pivoting into AI, I spent years crafting high-performance web platforms — from a real-time virtual classroom whiteboard with WebRTC to SEO-optimized NextJS apps serving millions of students.
